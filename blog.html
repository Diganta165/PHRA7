1. Differences among map(), filter(), forEach() and find()

Every single one of them is used to iterate through an array but map() returns value, forEach() doesn't return anything. filter() is used to get multiple array elements that full fills the condition but in the case of find() it returns only one array element that full fills the condition.

2. Differences among localStorage, sessionStorage and cookies

All of these are used to store data on the browser but localStorage's data stays unless you remove those data. In the case of Session storage the data gets removed when the browser or session gets closed. And for cookies it can store very limited amount of data like 4kb but the data can be set to expire at a certain time.

What is event loop? And how does it work?

Event loop is a process to execute the code, collecting and processing events and executing the queued tasks. Event loop works in a cycle where it continually checks whether or not the call stack is empty. And if so then a new function is added from the event queue. And this goes on and on again.
